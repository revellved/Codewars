# Codewars (TypeScript)

## STARTED

You don't need much to get started.

### Must have

- Node

### Optional

- TSserver (LSP)

## Commands

> RECOMENDATION: Install LSP server TSserver for your IDE 

- Install packages (npm)
    - npm install

- Run all tests 
  - npm run test

## PACKAGES

- @codewars/test-compat
- @types/chai
- @types/mocha
- chai
- mocha
- ts-node
- typescript
