# Codewars (C)

## STARTED

You don't need much to get started.

### Must have

- GCC or Clang
- Make

### Optional

- Clangd (LSP server)

## Commands

> RECOMENDATION: Install LSP server Clangd for your IDE 

- Run all test  
  - make gcc_test
    > NOTE: If you use GCC
  - make clang_test
    > NOTE: If you use Clang
